Analytics Components Preview

üöÄ Development Preview

This page allows you to preview all analytics components without running Jekyll.

To use:

  1. Run npm run dev in this directory
  2. Vite will serve this file and hot-reload on changes
  3. Components will auto-mount to their respective divs below

Population Chart

Log-log scatter plot comparing country population to unique website visits. Hover over points to see details.

Embedding code: <div id="population-chart-root"></div>
Script: <script type="module" src="/widgets/dist/population-chart.js"></script>

Geography Charts

Multiple visualizations showing YouTube views and site analytics across different countries. Includes views vs population, site uniques vs population, and relative engagement rates.

Embedding code: <div id="geography-charts-root"></div>
Script: <script type="module" src="/widgets/dist/geography-charts.js"></script>

Interactive Counter Widget

A simple React counter demonstrating state management and user interaction. Perfect for testing React embedding in blog posts.

Embedding code: <div id="counter-widget-root" data-initial="0" data-step="1"></div>
Script: <script type="module" src="/widgets/dist/counter-widget.js"></script>
Note: Use data-initial and data-step attributes to configure starting value and increment.

Following Eyes Widget

Interactive eyes that follow your cursor around the screen. Demonstrates event handling and real-time animations in React.

Embedding code: <div id="following-eyes-widget-root"></div>
Script: <script type="module" src="/widgets/dist/following-eyes-widget.js"></script>
